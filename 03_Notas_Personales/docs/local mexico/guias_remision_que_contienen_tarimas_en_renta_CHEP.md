# Integración de Tarimas CHEP

La integración de tarimas CHEP se base en una lambda privada en aws, la cual esta programada para ejecutarse todos los días, ejecuta un stored procudere en la MXBDAJE para obtener los datos en formato EDI de las tarimas y enviarlas al FTP del proveedor.

## 1. Procedimiento almacenado

Este procedimiento genera el listado de envíos (guías) de tarimas CHEP hacia clientes internos y externos. El formato de salida se encuentra en formato EDI tal como lo exige el proveedor para que sea recepcionado e integrado automáticamente por su servicio SFTP.

* `[USP_CHEP_GEN_FILE_MOVIMIENTOS]`
* `@P_COMPANIA CHAR(4),`
* `@P_SUCURSAL CHAR(4),`
* `@P_FECINI DATETIME,`
* `@P_FECFIN DATETIME`

### Parámetros:

* **@P_COMPANIA CHAR(4)** - Compañía requerida
* **@P_SUCURSAL CHAR(4)** - Sucursal
* **@P_FECINI DATETIME** - Fecha de inicio de búsqueda
* **@P_FECFIN DATETIME** - Fecha de fin de búsqueda

## Datos de prueba para ejecución de SP

`EXEC USP_CHEP_GEN_FILE_MOVIMIENTOS '0030', '0068', '03/06/2025', '03/06/2025'`
